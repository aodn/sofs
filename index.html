Real time plots of data from the Deep Water Moorings (DWM) Southern Ocean Flux Station


Data download and further information about the SOFS mooring are available from the IMOS Ocean Portal

Metadata for the SOFS mooring is available from the IMOS Metadata Catalogue

Only the last 7 days of data are displayed. Data plots are updated 4-hourly.



        All parameters are measured with two sets of instruments.  Values from logger L-22 are showin in blue. Values from logger L-23 are shown in green. Date/Time is in UTC and the format is YYYYMMDD.

SPD = Wind speed (m/s)
U = Eastward component of wind speed (m/s)

DIR = Wind direction (going to) (degrees)
V = Northward component of wind speed (m/s)

PRECIP = Precipitation amount (mm)
RAD_LW = Long wave radiation (W/m2)

RH = Relative humidity (%)
RAD_SW = Short wave radiation (W/m2)

T = Air temperature (°C)
P = Atmosphere pressure (mbar)

TS = Water temperature (°C)
COND = Conductivity (Siemens)
SAL = Salinity (PSU)
